# 進撃のこうかとん
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公こうかとんが仲間と協力し、強力なボスを倒してくシューティングゲーム

## ゲームの実装
###共通基本機能
* 主人公キャラクターに関するクラス Bird
* 仲間に関するクラス Bird2
* 敵に関するクラス Enemy
* 爆弾に関するクラス Bomb
* ビームに関するクラス Beam
* 仲間のビームに関するクラス Beam2
* 特殊ビームに関するクラス NeoBeam
* 爆破エフェクトのクラス Explosion
* スコアに関するクラス Score

### 担当追加機能
* 仲間召喚(担当：佐々木)：仲間がアシストしてくれるクラス
