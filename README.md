# ゲーム のタイトル
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公キャラクター(こうかとん)が仲間と協力し、強力なボスを倒していくシューティングゲーム

## ゲームの実装
###共通基本機能
* 主人公キャラクターに関するクラス
* 爆弾に関するクラス
* ビームに関するクラス
* 新しいビームに関するクラス
* 爆発に関するクラス
* 敵機に関するクラス
* スコアに関するクラス
### 担当追加機能
* ファイヤー出現
* 火にあたると七面鳥の丸焼き
